# Changelog

## 1.0.0 — 2026-05-01

Initial release.

- Per-service pricing (Name|Price format in settings)
- Stripe PaymentIntent integration with test/live mode toggle
- 3-step booking form with custom calendar and time slot picker
- Branded HTML email notifications (customer confirmation, admin alert, status updates)
- Admin dashboard with appointment list, status filtering, inline status changes
- Configurable business days, hours, blocked dates, min notice, max advance window
- Shortcode `[chm_booking]` for any page/builder
- Mobile-responsive booking UI
